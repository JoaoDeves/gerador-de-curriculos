# Gerador de Currículos

Projeto desenvolvido como prática acadêmica para criação de um gerador de currículos dinâmico utilizando HTML, PHP, CSS, Bootstrap e JavaScript.  
O usuário preenche um formulário com suas informações pessoais, formação acadêmica e experiência profissional, e o sistema gera automaticamente uma página pronta para impressão ou salvar como PDF.

---

## Funcionalidades

- Formulário intuitivo para preenchimento dos dados.
- Adição dinâmica de:
    - Formação acadêmica
    - Experiências profissionais
- Cálculo automático de idade pela data de nascimento.
- Máscara simples para telefone.
- Página final gerada automaticamente.
- Função integrada para Imprimir / Salvar como PDF.
- Layout responsivo utilizando Bootstrap.

---

## Tecnologias Utilizadas

| HTML5
| CSS3 + Bootstrap 5
| JavaScript / jQuery
| PHP

---


## Como Usar

1. Preencha o formulário com seus dados pessoais
2. A idade será calculada automaticamente ao informar a data de nascimento
3. Adicione formações academicas clicando no botão "+ Adicionar"
4. Adicione quantas experiências profissionais precisar clicando no botão "+ Adicionar"
5. Clique em "Gerar Currículo"
6. Use o botão "Imprimir / Salvar como PDF" para baixar seu currículo
7. No navegador, escolha "Salvar como PDF" na opção de impressão
